# adspower_rpa
Aplicação de RPA para o Adspower

## Novas Opções

O menu lateral da aplicação agora inclui duas automações adicionais:

- **🗺️ Google Maps Automation** – abre a página para fluxos de automação no Google Maps.
- **🔍 Gsearch Automation** – abre a página de automação para buscas no Google.
